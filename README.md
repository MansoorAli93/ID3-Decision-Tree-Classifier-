# ID3 Decision Tree Classifier 

## Project Overview
This project implements the **ID3 (Iterative Dichotomiser 3) Decision Tree algorithm** using Python and Streamlit.  
The application allows users to train a decision tree on categorical data and make predictions through an interactive web interface.

## Features
- Upload custom CSV dataset
- Automatic entropy and information gain calculation
- Decision Tree generation using ID3 algorithm
- Interactive prediction inputs
- Dataset preview toggle
- Clean and responsive Streamlit UI

## Resources Used
- Python
- Streamlit
- Pandas
- NumPy

## How It Works
1. Load default dataset or upload your own CSV file.
2. Select the target column.
3. Click **Train Model** to build the decision tree.
4. Provide feature inputs.
5. Click **Predict** to get the classification result.

## Purpose
This project demonstrates:
- Entropy calculation
- Information Gain
- Decision Tree construction
- Basic Machine Learning concepts
- Streamlit web deployment
